Bulk File Renamer is a Python script that automatically **renames files in bulk** and **organizes them into folders** based on their file types. It helps you manage large collections of images, documents, data files, and downloads efficiently.

## ‚ö° Features
- Rename multiple files automatically with a consistent naming scheme.

  - **Data:** `.csv`, `.xlsx`  
  - **Downloads:** `.download`  
  - **Others:** unknown file types  
- Safe testing with sample files.  
- Fully configurable for additional file types.
